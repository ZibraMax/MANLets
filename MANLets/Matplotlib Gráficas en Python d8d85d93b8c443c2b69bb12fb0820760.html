Varias gr√°ficas en una sola figura

En Matplotlib hay muchas formas de hacer subplots,. que b√°sicamente son distintas gr√°ficas en una sola imagen. Para ello hay que hacer la gr√°fica directamente en cuadros que esten contenidos en una figura (algo extra√±o). Por ejemplo

ax.set_title('Gr√°fica 1') #Para poner t√≠tulos y nombres de ejes hay que usar estos comandos

#Como es una grilla de 1x3 gr√°ficas falta agregar las otras dos. Esto se hace igual

Gr√°ficas en 3D

Las gr√°fias en 3D pueden generarse de distintas maneras. En la p√°gina https://matplotlib.org/mpl_toolkits/mplot3d/tutorial.html hay una referencia completa de todas las opciones.

Para las gr√°ficas en 3D hay que importar una parte adicional de matplotlib que es:

from mpl_toolkits.mplot3d import Axes3D

Para este ejemplo usaremos la funci√≥n 2 y crearemos los datos as√≠:

X = []
Y = []
Z = []

_x = np.linspace(x0,xf,n)

for i in range(n):
    for j in range(n):
        X.append(_x[i])
        Y.append(_x[j])
        Z.append(f2(_x[i],_x[j],3))

Esto genera una serie de puntos X,Y,Z donde pasar√° la superficie.

Superficies

Para este caso realizaremos una superficie de triangulos que es la manera mas facil de crear una superficie tridimensional.

Para crear superficies hay que usar figuras de matplotlib as√≠:

fig = plt.figure()

ax = fig.add_subplot(projection='3d') #Decimos que es uan gr√°fica en 3D

surf = ax.plot_trisurf(X,Y,Z,cmap='magma') ##Graficamos la superficie

fig.colorbar(surf) #Le agregamos una barra de color
plt.show()

El CMAP es el c√≥digo de color que usar√° matplotlib para indicar la altura. Hay una gran cantidad de colormaps por defecto: https://matplotlib.org/stable/tutorials/colors/colormaps.html

Como son gr√°ficas que estan dentro de figuras, pueden combinarse para crear combinaciones interesantes
