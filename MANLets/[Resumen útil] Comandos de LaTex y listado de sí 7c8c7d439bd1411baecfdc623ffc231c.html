Fracciones

Para hacer fracciones se puede usar el símbolo / (a/ba/ba/b﻿) o la sentencia

\frac{a}{b}

Raices

Se pueden usar recursivamente y se puede asignar cualquier exponente

\frac{\sqrt[3]{x}}{A}

Logaritmos

Se usa el símbolo \ para decirle a LaTex que es una palabra especuial

\log_{10}(35)

\ln(35)

Funciones trigonométricas

Se recrean usando índices y subíndices.

\frac{\cos^{-1}(x)}{x^{\sqrt{x}}}

En el ejemplo anterior se muestra una ecuación que en el denominador tiene una raiz como exponente, por lo tanto, cualquier cosa que se ponga dentro de los parentesis { } se renderizará como el índice o subíndice de la expresión. Esta es una de las claves para poder escribir cualquier ecuación con LaTex.

Sumatorias

Crea una sumatoria con índice de inicio y final. Estos índices pueden ser omitidos si se requiere 

\sum^{i=0}_{n}{i}

Productorias

Crea una productoria con índice de inicio y final. Estos índices pueden ser omitidos si se requiere 

\prod^{i=1}_{n}{i}

\coprod^{i=1}_{n}{i}

Integrales

Crea una integral con límites de inicio y final. Estos límites pueden ser omitidos si se requiere 

\int_{a}^{b}{e^x}{dx}

Se pueden poner una detras de otra para generar integrales múltiples

\int_{a}^{b}{\int_{c}^{d}{\frac{e^x}{y}}{dy}}{dx}

\oint_{a}^{b}{e^x}{dx}
