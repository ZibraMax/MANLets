Las variables son la manera mas sencilla de almacenar informaci√≥n. Existen muchos tipos de variables que tienen diferentes caracter√≠sticas. Por ejemplo, las variables de tipo int sirven para almacenar n√∫meros enteros, mientras que las variables de tipo float o double sirven para amacenar n√∫meros con decimales.

Tener variables nos permite usar la informaci√≥n y hacer cambios en ella. Por ejemplo, si tenemos dos variables que representen n√∫meros podr√≠amos asignar la suma de estas dos variables a una variable nueva.

Las variables representan un espacio en la memoria del computador a la que se puede acceder mediante un nombre, sin embargo, a nivel de usuario esto no es relevante, ya que el lenguanje de programaci√≥n se encargar√° de hacer estos procesos.

En conclusi√≥n, las variables al nivel de usario tendr√°n: Un nombre, un tipo y un valor.

El funcionamiento del While es parecido al for, sin embargo, su diferencia radica en que el for hace un n√∫mero de iteraciones predefinido, mientras que el while itera dependiendo de una condici√≥n booleana (muy parecido al if)

Por ejemplo, un ciclo while que se termine cuando el n√∫mero es divisible entre 7.

lista = [3,6,3,8,32,27,28,65,34]

	i = i + 1

Otra manera de hacerlo:

Si los Whiles estan mal formulados pueden llevar a un bucle infinito en el programa, por lo que deben tratarse con precauci√≥n.

Las funciones se declaran usando la palabra def , con la siguiente sintaxis:

Hagamos una funci√≥n que nos diga si un n√∫mero es par o impar

Luego tenemos que evaluar de manera num√©rica si el n√∫mero es o no par, por ejemplo, podemos usar la operaci√≥n m√≥dulo, porque, todo n√∫mero par es divisible entre 2, lo que hace que el residuo sea 0.

>>> None

La funci√≥n no retorna nada, por eso al imprimir sale None

Por √∫tlimo debemos retornar el booleano, esto lo podemos evaluar con un if

Note que pusimos el return dentro del if, esto tambien es v√°lido.
